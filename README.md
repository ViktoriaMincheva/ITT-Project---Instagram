## Main functionalities
-  Register/Login
-  Profile page
-  Edit profile
-  See other users profile
-  Follow/Unfollow users
-  Search by user
-  Chat functionality
-  Upload photo
-  Upload story
-  See all stories
-  Explore all posts
-  Like/Dislike posts
-  Save/Unsave posts
-  Comment post
-  Delete post
